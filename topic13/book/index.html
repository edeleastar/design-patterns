Setup

In XCode, create a new 'Empty' Swift application:

Make sure the app is a Swift application:

Run the app now using the simulator:

You should have a single source file:

import UIKit

@UIApplicationMain
class AppDelegate: UIResponder, UIApplicationDelegate
{
  var window: UIWindow?

  func application(application: UIApplication, didFinishLaunchingWithOptions launchOptions: NSDictionary?) -> Bool
  {
    self.window = UIWindow(frame: UIScreen.mainScreen().bounds)
    // Override point for customization after application launch.
    self.window!.backgroundColor = UIColor.whiteColor()
    self.window!.makeKeyAndVisible()
    return true
  }

  func applicationWillResignActive(application: UIApplication)
  {
  }

  func applicationDidEnterBackground(application: UIApplication)
  {
  }

  func applicationWillEnterForeground(application: UIApplication)
  {
  }

  func applicationDidBecomeActive(application: UIApplication)
  {
  }

  func applicationWillTerminate(application: UIApplication)
  {
  }
}

Open the Storyboard - which will be blank - and drag a ViewController from the Object Library onto the canvas:

Back in project settings - make sure the Main storyboard we have created is set as the `Main Interface'

Try running the app. Does the Label appear?

No - it is not visible yet. We need to edit the AppDelegate.swift file, and change this method:

  func application(application: UIApplication, didFinishLaunchingWithOptions launchOptions: NSDictionary?) -> Bool
  {
    return true
  }

(We have removed some default initialization)

Run the app now, and the Label should appear.
